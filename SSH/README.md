# SSH Configuration (Cisco Router)

This folder contains SSH configuration files created and tested in Cisco Packet Tracer using CLI mode.  
It includes step-by-step commands for secure remote access setup on Cisco routers.

## Files Included
- **SSH_Configuration.pkt** – Complete Cisco Packet Tracer project file.  
- **SSH_Configuration.docx** – Document explaining all CLI commands and configuration details.

## Key Topics
- SSH Setup and Verification  
- Enabling Secure Remote Access  
- User Authentication Configuration
